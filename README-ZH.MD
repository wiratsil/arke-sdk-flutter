## IDE

- 安装 [Android Studio](https://developer.android.google.cn/studio/index.html)，Android SDK API 25。

## 设备环境要求

- 设备中存在 `domain_cfg` 和 `domain_enable` 模块，只需安装 USDK apk 即可。[下载地址](http://developer.landicorp.com/device-maintenance/usdk-release)
- 设备中不存在 `domain_enable` 模块，有两种情况：
  - 如果需要使用 `2.1.3` 版本之前的 USDK，需要安装包含权限包和 USDK apk 的 UNS 文件。[下载地址](http://developer.landicorp.com/device-maintenance/usdk-beta)
  - 如果使用 `2.1.3` 版本之后的 USDK，只需安装 USDK apk 即可。[下载地址](http://developer.landicorp.com/device-maintenance/usdk-release)
